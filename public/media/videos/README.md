# Videos Folder

Add your videos here!

**Supported formats:** MP4, WebM, MOV

**How it works:**
- Add videos to this folder
- Name them: `video-1.mp4`, `video-2.mp4`, etc.
- Or use descriptive names: `anniversary-moment.mp4`
- Videos will appear in the gallery with a play button
- Edit captions using the Admin Panel (press `Ctrl+K` or click the edit icon)

**Tips:**
- Recommended format: MP4 (H.264 codec)
- Keep file sizes reasonable (under 50MB each)
- Videos will autoplay on hover (muted) and play with sound on click
