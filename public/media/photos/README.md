# Photos Folder

Add your photos here! 

**Supported formats:** JPG, JPEG, PNG, GIF, WebP

**How it works:**
- Add photos to this folder
- Name them in order: `photo-1.jpg`, `photo-2.jpg`, etc.
- Or use descriptive names: `first-date.jpg`, `beach-picnic.jpg`
- The website will automatically detect and display them
- Edit captions using the Admin Panel (press `Ctrl+K` or click the edit icon)

**Tips:**
- Recommended size: 1200x800px or larger
- Keep file sizes reasonable (under 2MB each)
- The website will automatically optimize and display them beautifully
